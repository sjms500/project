git

  • 깃허브 라파지토리 설정
  • http://git-scm.com/ 다운로드 받아서 설치
  • 윈도우 창에서 git bash 열기
  • $ git config --global user.name "sjms500"
  • $ git config --global user.email "sjms500@naver.com"
  • $ git config --list
  • 비쥬얼 스튜디오 터미널 실행
  • +클릭 후 git bash 선택
  • 터미널에서 마스터 확인 후
  • git init 초기화는 한번만
  • git add . 파일 전체 업로드
  • git status 상태확인 (안해도 됨)
  • git commit -m "first commit" 히스토리 위한 메세지 작성 같으면 오류남
  • git remote add origin https://github.com/sjms500/project.git 깃에 링크 주소 복사 붙여넣기(로컬과 깃연결 위해 repository 주소 복사)
  • git remote -v 잘 연결 되었는지 확인
  • git push origin master 업로드
